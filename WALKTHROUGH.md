## Key Commands

### Game Basics
- **LOOK** - Examine your surroundings
- **INVENTORY** (or **I**) - Check what you're carrying
- **TAKE [item]** - Pick up an object
- **DROP [item]** - Put down an object
- **EXAMINE [item/person]** - Get detailed information
- **GO [direction]** - Move (NORTH, SOUTH, EAST, WEST, UP, DOWN)
- **STATUS** (or **PROGRESS** or **METRICS**) - View your progress dashboard

### Interacting with NPCs
- **TELL [person]** - Talk to team members (while holding relevant items!)
- **EXAMINE [person]** - Get hints about what they need

### Puzzle Solving
- **PUT [item] ON [mainframe]** - Install strategy components
- **PUSH [screen]** or **TURN ON [screen]** - Start your presentation
